# Movie_recommendation_system
#This project was implemented while following a course/tutorial. I recreated the project to understand the concepts of Machine Learning.
#Recommend movies to users based on similarity
#The project uses the TMDB 5000 movie dataset
#various tools are used in this project like python (used for data analysis, machine learning, recommendation system), jupyter notebook(used for writing code, showing output, visualization, documentation), pandas(used for reading datasets, filtering data, merging tables, cleaning data), numpy (helps in matrix operations, vector calculations, similarity calculations), scikit-learn(used for text vectorization, similarity calculation), countvectorizer(converts text to numbers), NLP(used for finding similarity among different movies), cosine similarity(it measures how similar two movies are).
# Steps of the Data science pipeline in this project : 1. Import libraries, 2. Load dataset, 3. Data cleaning, 4. Merge datsets, 5. Feature selection, 6. Feature engineering, 7. Text vectorization, 8. Calculate similarity, 9. Recommendation system.
